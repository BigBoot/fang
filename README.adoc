= Fang
 
== How to build

Windows

 gradlew.bat shadowJar
 
Linux

 ./gradlew shadowJar
 
The resulting jar file will be located in: `build/libs/fang-1.0-SNAPSHOT-all.jar`

== How to Run

 java -jar build/libs/fang-1.0-SNAPSHOT-all.jar
